## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

# create entreprise

$ POST ===> api/settings
$ form Data ===> {

name: entreprise name
description: entreprise description
firstname: admin firstname
lastname: admin lastname
contact: admin contact
password: admin password
email: admin email

}

````






## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
````
